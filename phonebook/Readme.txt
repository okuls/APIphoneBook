Для запуска API:
1. Запустить файл createDb.py (создаётся БД с полями)
2. Запустить файл restful.py (API запустится на локальном сервере на 127.0.0.1:80)
3. Открыть в катологе с API index.html (или запустить другой локальный сервер на другом порту)


1. Для получения контакта необходимо:
Отправить GET запрос по адресу http://127.0.0.1:80/contact/
форма ответа будет json со структурой:
{name: "Имя", phone: 89886665544},
где 	name - это имя контакта,
	phone - это номер телефона.

2. Для записи контакта необходимо:
Отправить POST запрос по адресу http://127.0.0.1:80/contact/
Объект должен быть формата json
Пример объекта {"name": "Имя", "phone": 89778776655}
где	name - это имя контакта,
	phone - это номер телефона в формате int.

В ответ будет вернётся объект формата {"result": result}
В поле result будет написан результат работы:
Варианты:
	{"result": "Имя не указано!"}
	{"result": "Телефон указан некорректно! Формат: 89778776655"}
	{"result": "Контакт с таким именем уже существует!"}
	{"result": "Контакт успешно добавлен."}
